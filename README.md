# 🔍 GEO Score Tool

**GEO Score Tool** è una web app professionale progettata per agenzie e consulenti SEO. Il tool analizza l'ottimizzazione di una pagina web per la **Generative Engine Optimization (GEO)**, valutando quanto i contenuti siano pronti per essere citati dai motori di ricerca basati su AI (ChatGPT, Perplexity, Claude).

---

## 🚀 Funzionalità Principali
* **Audit Multidimensionale:** Valutazione automatica su 5 pilastri (Clarity, Structure, Entity, Trust, Tech).
* **GEO Score™:** Un punteggio da 0 a 100 con classificazione del livello di "AI Readiness".
* **Suggerimenti Generativi:** Generazione automatica di FAQ, H2 e Schema Markup personalizzati.
* **Report per Agenzia:** Sezioni dedicate con problemi prioritari e quick wins.
